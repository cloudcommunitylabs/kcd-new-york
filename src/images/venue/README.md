# Venue Images

This directory contains photos of Convene One Liberty Plaza, the venue for KCD New York 2026.

## Photos Needed

1. **Exterior shot** (1 photo)
   - Building entrance or street view
   - Shows location context

2. **Interior/Conference room** (2-3 photos)
   - Main conference space
   - Stage setup
   - Seating arrangement

3. **Networking space** (1 photo)
   - Common areas
   - Break-out spaces

## Image Requirements

### Processing
- **Resize**: 1920x1280px max
- **Format**: JPEG with WebP via Gatsby
- **Compression**: 80% quality

### File Naming
Format: `convene-{area}-{number}.jpg`

Examples:
- `convene-exterior-01.jpg`
- `convene-interior-01.jpg`
- `convene-conference-room-01.jpg`
- `convene-networking-space-01.jpg`
